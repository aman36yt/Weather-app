# 🌦️ Weather Web App

<p align="center">
  <b>A responsive weather application that fetches real-time weather data using an API 🌍</b>
</p>

---

## ✨ Features

🌡️ Get real-time weather information
🔍 Search weather by city name
📍 Displays temperature, weather condition, and more
⚡ Fast API integration
📱 Responsive design for all devices
🎨 Clean and modern UI

---

## 🛠️ Tech Stack

* 🧱 HTML5 – Structure
* 🎨 CSS3 – Styling
* ⚡ JavaScript – Logic
* 🌐 API – Fetch weather data

---

## ⚙️ How It Works

* User enters a city name
* App sends a request to the weather API
* Receives real-time data
* Displays temperature, weather condition, etc.
